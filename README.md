#  Mammura AI - Breast Cancer Detection System

##  Overview
Mammura AI is an intelligent deep learning-based web application designed to assist in early breast cancer detection using medical imaging.

It analyzes images and classifies them as:
- ❌ Malignant (Cancer)
- ✅ Benign (Non-Cancer)

---

##  Dataset
The model is trained on the **Breast Histopathology Images (IDC) dataset** from Kaggle.

- 📂 Total Images: ~2,77,000+
-  Classes:
  - 0 → Benign (Non-Cancer)
  - 1 → Malignant (Cancer)
-  Images resized to **128x128 pixels** for training

This dataset contains real microscopic breast tissue images used for cancer detection.

🔗 Dataset Link:  
https://www.kaggle.com/datasets/paultimothymooney/breast-histopathology-images

---

##  Features
-  AI-powered image classification  
-  Image upload with preview  
-  Confidence-based prediction  
-  Modern and interactive UI  
-  Fast Flask backend  

---

##  Tech Stack
- Python  
- TensorFlow / Keras  
- Flask  
- HTML, CSS, JavaScript  

---

##  How It Works
1. User uploads a medical image  
2. Image is preprocessed (resized & normalized)  
3. Deep learning model analyzes the image  
4. Output is displayed with prediction and confidence score
